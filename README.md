# BokBok - Books Exchange Platform

## The Project
The project focuses on creating an online platform for **book exchanges** between users.
Book data such as title, author(s), and publication date are retrieved from the ***Google Books API*** (up to 1,000 free requests per day).

## Repository Contents

**Jupyter Notebook:** contains all the code required to develop and run the platform. 

**PDF Report:** explains the development process and highlights the key features and specific aspects of the project.
